Behold: The Ugliest UI

A static site demonstrating an intentionally awful UI and how to improve it.

Screenshot of an ugly UI

Why this UI is bad

  • Horrible color clashes and unreadable text.
  • Inconsistent spacing and alignment.
  • Overuse of different font sizes and decorations.
  • No clear visual hierarchy or affordances.
  • Poor accessibility (contrast, no alt text for content, tiny clickable targets).

How I would improve it

  1. Choose a limited, harmonious color palette (2–3 colors plus neutrals) and ensure accessible contrast ratios.
  2. Establish a consistent typographic scale (font sizes, line-height) and stick to 1–2 web-safe fonts or variable fonts.
  3. Create a clear layout grid for alignment and spacing; use consistent padding/margins and group related items.
  4. Improve affordances: use clear buttons, consistent primary/secondary actions, and visible focus states for keyboard users.
  5. Run accessibility checks (WCAG) and automated linters, and test with keyboard and screen readers.
